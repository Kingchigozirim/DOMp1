# Shopping Cart - DOM Workshop

A simple, interactive **Shopping Cart** built with HTML, CSS, and vanilla JavaScript — perfect for learning DOM manipulation.

Users can:
- Increase / decrease item quantity using **+** and **−** buttons
- Remove items from the cart using the trash icon
- Like / unlike products using the heart icon (changes to red when liked)
- See the **total price** update in real-time after every action

## Features

- Quantity adjustment (± buttons)
- Item deletion (trash icon removes product completely)
- Like functionality (heart turns red when liked, black when unliked)
- Dynamic total price calculation
- Clean and responsive layout using Bootstrap 5
- Font Awesome icons for intuitive controls

## Technologies Used

- **HTML5**
- **CSS3** (custom styles + Bootstrap 5.1.3)
- **JavaScript** (vanilla — no frameworks)
- **Bootstrap 5.1.3** (via CDN)
- **Font Awesome 6** (via CDN)
